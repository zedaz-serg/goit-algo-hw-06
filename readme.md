# Обґрунтування різниці
## Алгоритм BFS (Пошук у ширину):
- Принцип: Досліджує граф пошарово, гарантуючи, що всі вершини на відстані k будуть відвідані перед вершинами на відстані k+1.
- Шлях: Завжди знаходить найкоротший шлях у незваженому графі.
- Результат: Шлях через Золоті Ворота має 13 ребер.
## Алгоритм DFS (Пошук у глибину):
- Принцип: Йде максимально глибоко по одній гілці, доки не знайде ціль.
- Шлях: Не гарантує найкоротшого шляху. Через те, що в цій моделі обидва шляхи переходу (Театральна <-> Золоті Ворота та Хрещатик <-> Майдан Незалежності) є короткими, і обидва ведуть до Палацу Спорту, у даному конкретному випадку DFS міг знайти шлях такої ж довжини (або навіть довший/коротший) залежно від внутрішнього порядку перебору сусідів.
- Результат: Шлях через Хрещатик має 13 ребер.
# У цьому конкретному випадку обидва шляхи мають однакову довжину.
